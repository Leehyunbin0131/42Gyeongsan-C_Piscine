# 42Gyeongsan C Piscine

This repository contains my C Piscine exercises from 42 Gyeongsan.

The Piscine is an intensive introduction to C programming, Unix basics, and
problem solving through small exercises. Each directory follows the original
42 Piscine subject structure.

## Contents

| Module | Topics |
| --- | --- |
| `C_Piscine_C_00` | Basic output, loops, and number printing |
| `C_Piscine_C_01` | Pointers, arrays, string output, and integer arrays |
| `C_Piscine_C_02` | String copy/check/transform functions |
| `C_Piscine_C_03` | String comparison, concatenation, and search |
| `C_Piscine_C_04` | String length, number output, `atoi`, and base conversion |
| `C_Piscine_C_05` | Recursion, powers, Fibonacci, square root, and prime numbers |
| `C_Piscine_C_06` | Command-line arguments |
| `C_Piscine_C_07` | Dynamic allocation and string joining |
| `C_Piscine_C_08` | Header files and macros |

## Notes

These exercises are kept as a learning archive. The code follows the constraints
of the original 42 Piscine subjects, so some standard library functions are not
used even when they would normally be available in a C project.
